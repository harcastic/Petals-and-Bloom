# FlowerSite
Petals-and-Bloom

A modern, professional flower e-commerce website frontend built with HTML, CSS, and JavaScript.

## Features
- Beautiful product cards
- Shopping cart functionality
- Responsive design
- Animated buttons and transitions
- Clean, user-friendly interface

## Technologies Used
- HTML5
- CSS3 (custom properties, transitions, animations)
- JavaScript (dynamic content, cart logic)
- Font Awesome (icons)

## Getting Started
1. Clone the repository:
   ```
   git clone https://github.com/harcastic/Petals-and-Bloom.git
   ```
2. Open `index.html` in your browser.

## File Structure
- `index.html` — Main site structure
- `styles.css` — All styling and animations
- `script.js` — Interactivity and cart logic

## Customization
- Update product data in `script.js` to add or modify flowers.
- Adjust styles in `styles.css` for branding.
